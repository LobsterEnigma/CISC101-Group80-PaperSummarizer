# Research Paper Summarizer Project
**Group Number:** [80]
**Course:** CISC 101

## Project Overview
This repository contains the System Prompt and Modular Architecture for a Research Paper Summarizer application, generated using Microsoft Copilot.

## Repository Components
Here is an explanation of the files in this repository:

* **`system_prompt.md`**: The main instruction file that defines the persona, inputs, outputs, and constraints for the AI summarizer.
* **`modules/`**: A folder have 6 specific functional modules:
  * `01_intake.md`: Handles paper input and normalization.
  * `02_section_loop.md`: Summarizes each section individually.
  * `03_guardrails.md`: Ensures safety and checks for hallucinations.
  * `04_rendering.md`: Formats the final output table.
  * `05_key_contributions.md`: Extracts core contributions .
  * `06_citation_extractor.md`: Extracts references .
* **`Verification paragraph and Text-based flowchart.md`**: Contains the verification proof and the system flowchart for Task 1.2.
* **`screenshots/`**: screenshots about Copilot generation process.
